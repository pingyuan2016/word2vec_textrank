# 简介
1、本文中第一种方法是Word2vec+textrank，针对段落进行提取<br>
2、第二种利用textrank4zh工具提取关键句，见脚本textrank4zh.py
## 安装环境
math<br>
numpy<br>
jieba<br>
gensim<br>
networkx<br>
itertools<br>
textrank4zh<br>

## 摘要生成
本方法的摘要生成是抽取式，通过把段落训练词向量，得到句子的向量。再通过pagerank方法得到权重高的向量，从而得到对应的句子
